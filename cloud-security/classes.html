The industry-leading credential covers the full AI lifecycle from generative AI fundamentals and architecture to governance, risk management, privacy, and cloud security.

  • Unique, innovative and well-defined content

CCSK - Certificate of Cloud Security Knowledge

Provider: Cloud Security Alliance

Foundation

Certificate of Cloud Security Knowledge (CCSK) was created by the Cloud Security Alliance in 2010 to address the growing need for security knowledge around cloud computing. CCSK certification covers all major aspects in information security including risk management, provider evaluation, infrastructure and application security, identity management, and highlights the recommended approaches and tools required for secure cloud adoption.

CCSK Foundation (2 days)

Frontal teaching of 6 modules covering all topics relevant to cloud security. Contains all information needed for passing the CCSK exam. Exam voucher included.

CCSK Plus (+1 day)

Additional day devoted to practicing cloud labs. Students practice different security tools available for IaaS/PaaS environments.

Pricing:

  • Without labs: $995 per student
  • With labs: $1,695 per student

Includes class kit with books, labs, and two vouchers for the CCSK exam. Minimum 5 attendees.

CCZT - Certificate of Competence in Zero Trust

Provider: Cloud Security Alliance

Zero Trust

The Certificate of Competence in Zero Trust (CCZT) is the industry's first, most authoritative, vendor-neutral Zero Trust training and certificate program that delivers mainstream best practices for developing and implementing a Zero Trust philosophy. Includes foundational components from CISA and NIST, innovative work in Software Defined Perimeter by CSA Research, and guidance from renowned experts including John Kindervag, Founder of the Zero Trust philosophy.

Zero Trust Foundational Concepts

Definitions, goals, objectives and benefits

Zero Trust Architecture

Logical models, key components, workflows

Software Defined Perimeter

Core tenets, technologies, and benefits

Zero Trust Planning

Stakeholder identification, protect surfaces

Zero Trust Implementation

Network setup, agents, automation

Industry Best Practices

CISA, NIST SP 800-207 guidance

Price: $1,295 per student

Includes class kit with books. Minimum 5 attendees.
